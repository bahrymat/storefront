easyStorefront: Connecting businesses to the online world

How to run

  1. Start running a mongo server instance on port 8081.
  2. Install the node modules 'mongoose', 'validator' and 'multiparty', if necessary. Both are available using npm.
  3. Run the server using 'node run_server.js'.
  4. Open your web browser and navigate to localhost:8080.

As with assignment 2, the dump folder contains a mongodb dump that can be restored to add four example stores to the example store page. These have been updated to the new database format, however the stores themselves are unchanged from assignment 2.
